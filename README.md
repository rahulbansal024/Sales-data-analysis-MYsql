# Sales Data Analysis using MySQL

## Project Overview
This project analyzes sales data using MySQL and SQL queries to generate business insights.

## Technologies Used
- MySQL
- SQL

## Features
- Customer analysis
- Product sales analysis
- Revenue calculation
- Top-selling products
- Order management

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOIN
- Aggregate Functions

## Project Purpose
The goal of this project is to practice SQL queries, relational database management, and data analysis techniques.
